INTRODUCTION
============
This project contains samples for Listing Recommendations Service API developed in eclipse using the PHP Version 5.6.3

INSTRUCTIONS
============
Please make sure that you have JDK 1.6 or higher.

1. Unzip the file to the location of your choice.
2. Open eclipse workspace and import the project.
3. Configure the Build Path with the folder to the project classpath
5. Start the Apache Server 
4. Now run each sample as a PHP Web Application.


REFERENCES
==========
More information about this API can be found at : http://developer.ebay.com/Devzone/listing-recommendation/index.html
